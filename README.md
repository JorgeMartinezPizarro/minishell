### Commandos

Usa

```sh
make run
```

to compile and run the latest version of minitalk.


Con

```sh
make jorge "comando"
```

se ejecutara el comando "comando"

Con 

```sh
make jorge
```

se abre la shell version de jorge, no acepta tokens especiales, solo uso de variables, cd, env, echo y pwd de momento.
